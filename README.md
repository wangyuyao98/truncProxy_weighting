# Proximal Survival Analysis for Dependent Left Truncation

This repository contains implementation of the proposed estimators and simulations for the paper 
"Proximal Survival Analysis for Dependent Left Truncation" by Yuyao Wang, Andrew Ying, and Ronghui Xu. 

 - Folder `src`: functions for implementing the estimators.

 - `main_simu3.R`: main code for simulation.
